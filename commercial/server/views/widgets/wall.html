Wall widget

maxmert — issues and messages.
An element with height: 100%. It can use image or video as a background. Or add carousel inside. Useful in sites with big blocks of information. Inludes navigation.

Common usage

Description

To use wall just add -wall to the container.

CSS

<div class="-wall">
	<!-- Wall content here -->
</div>

JS

To activate the wall you can just add data attributes to the wall container. 
<div class="-wall" data-kind="wall" data-group="wall-group-name" data-name="current-wall-name">
	<!-- Wall content here -->
</div>
data attribute default description
kind wall JS will look for data-kind="wall" to initialize it as a wall.
group none JS will look for navigation bar with data-kind="wall-nav" and data-group="this-group-name" to add a bullet there. You can see the navigation bar on the right now.

To add a navigation bar simply put this html code in your DOM (not to the -wall) 

<div class="-nav-bullets _right_" data-kind="wall-nav" data-group="wall1"></div>
