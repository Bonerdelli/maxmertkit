Fb spinner

Variables

$spinner-fb-class: "#{$spinner-class}-fb" $spinner-fb-size: 30px $spinner-fb-max-addons: 10

Just add class -spinner-fb.This is a fb-like spinner.

You need to add -addon elements inside this spinner. The quantity is limited by sass variable $spinner-fb-max-addons.
