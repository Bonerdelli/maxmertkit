Global variables

$shader-class: "-shader" !global $shader-z-index: 100 !global

Shader is a background element for any content, for example a modal window.

Hidden by default. Add modifier _active_ to show up.
Dont't have any sizes.
Use modifier _responsive_ to make shader position: fixed.
By default shader is position: absolute
